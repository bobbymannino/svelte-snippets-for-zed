# Svelte Snippets for Zed

## Installation

1. Clone this repo:

```
git clone https://github.com/bobbymannino/svelte-snippets-for-zed
```

2. Go to Extensions menu in Zed IDE
3. Click "Install Dev Extension"
4. Select the folder you cloned
